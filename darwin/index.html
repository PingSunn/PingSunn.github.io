Brute Force Algorithm

เป็นเรื่องที่สำคัญและเป็นพื้นฐานของกรอบแนวความคิดทฤษฎีต่าง ๆ ของAlgorithm ใน Brute-force algorithm ปัญหาจะได้รับการแก้ไขด้วยวิธีที่ตรงไปตรงมาบนพื้นฐานของปัญหาและข้อกำหนดของปัญหา Brute-force algorithm นั้นออกแบบมาเพื่อที่จะแก้ปัญหาโดยไม่คำนึงถึงทรัพยากรของเครื่องคอมพิวเตอร์ที่ต้องใช้ อาทิเช่น ในบางครั้งการนำ Brute-force algorithm มาแก้ปัญหานั้นจะทำได้โดยการหาวิธีทุกวิธีที่ความเป็นไปได้ที่สามารถทำได้และหาคำตอบที่ดีที่สุด โดยทั่วไปแล้ว Brute-force algorithm นั้นเป็นการแก้ปัญหาที่ไร้เดียงสาที่ไม่ใช้โครงสร้างพิเศษ

ในการหาจำนวนสูงสุดในลำดับนั้นก็เป็น Brute-force algorithm เช่นกันเพราะเป็นการตรวจสอบแต่ละจำนวน n ในลำดับเพื่อหาจำนวนสูงสุด หรือจะเป็น Algorithm การหาผลรวมของจำนวน n จากการบวกเลขทีละครั้งก็ถือว่าเป็น Brute-force algorithm เช่นกัน Algorithm การหาผลคูณของเมทริกที่อยู่บนพื้นฐานของข้อกำหนด bubble sort , insertion sort และ selection sort ก็ถือว่าเป็น Brute-force algorithm ทั้งสาม Algorithm การเรียงลำดับนี้เป็นการหาอย่างตรงไปตรงมาที่ประสิทธิภาพน้อยกว่า Algorithm อื่น ๆ เช่น merge sort และ quick sort

แม้ว่า Brute-force algorithm จะไม่ค่อยมีประสิทธิภาพแต่ก็มีประโยชน์ Brute-force algorithm นั้นสามารถใช้ในการแก้ปัญหาเชิงปฏิบัติได้ โดยเฉพาะเมื่อรับค่าที่ไม่มากไป แม้ว่าจะไม่สามารถ Algorithm นี้กับการรับค่าจำนวนมาก นอกจากนี้การออกแบบ Algorithm เพื่อหาปัญหานั้น เป้าหมายก็เพื่อหา Algorithm ใหม่ที่มีประสิทธิภาพมากกว่า Brute-force algorithm
